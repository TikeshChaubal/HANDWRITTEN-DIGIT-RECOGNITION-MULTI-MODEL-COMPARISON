# HANDWRITTEN-DIGIT-RECOGNITION-MULTI-MODEL-COMPARISON
Evaluated six ML classifiers on the Kaggle MNIST dataset (784 features). Non-Linear SVM (RBF) and XGBoost achieved top accuracy (~97–98%), while Random Forest provided a fast, strong ensemble alternative. Decision Trees underperformed due to overfitting. Top models showed excellent class separability (ROC-AUC > 0.99). Predictions exported.
